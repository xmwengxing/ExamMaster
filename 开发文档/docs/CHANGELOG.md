# 更新日志

## [1.2.0] - 2026-01-07

### 新增功能
- **题库批量导入优化**
  - 支持2-8个选项（原固定4个）
  - 支持题干和解析中包含逗号等特殊字符
  - 增强CSV解析，使用正则表达式处理引号包裹字段
  - 完善数据验证机制（题型、题干、选项、答案）
  - 优化错误提示，显示具体行号和原因
  - 部分成功时询问是否继续导入

- **SSL证书配置**
  - 添加Cloudflare SSL配置指南
  - 支持Flexible SSL（快速方案）
  - 支持Origin Certificate（完整加密）
  - 提供自动配置脚本

### 优化改进
- **移动端底部导航**
  - 修复底部导航栏被内容遮挡问题
  - 优化移动端布局和间距

- **题库管理**
  - 添加题库等级字段
  - 优化题库卡片显示
  - 改进题库选择界面

- **管理员权限**
  - 完善管理员角色系统
  - 优化权限管理界面

- **实操练习**
  - 添加实操历史记录功能
  - 优化实操题目显示

- **练习模式**
  - 优化练习模式性能
  - 改进答题体验

### 修复问题
- 修复管理员页面空白问题
- 修复题库选择显示异常
- 修复考试发布功能
- 修复考试历史记录
- 修复督导统计数据显示
- 修复端口配置冲突

### 技术改进
- DeepSeek API迁移完成
- 优化数据库查询性能
- 改进错误处理机制
- 完善日志记录
